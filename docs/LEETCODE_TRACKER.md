## 📝 HOW TO USE THIS TRACKER

### Daily Routine:
1. ✅ Pick problem from current category
2. ⏱️ Set timer (45 min for Easy/Medium, 60+ for Hard)
3. 💻 Code solution in Java
4. 🧪 Test with edge cases
5. ✍️ Update status: ⬜ → 🟡 → ✅
6. 📝 Write notes in "Notes" column

### Status Legend:
- ⬜ = Not started
- 🟡 = In progress / Attempted
- ✅ = Solved (✓ after 3 days if re-solved)
- ⭐ = Optimal solution found

### Marking Complete:
When you solve a problem:
1. Change ⬜ to 🟡 immediately
2. After verifying correctness, change to ✅
3. After 3 days, re-solve and mark with ⭐ if optimal
